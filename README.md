# Sales Forecasting using Machine Learning

## Project Overview

This project predicts future sales using historical sales data and a Linear Regression model.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook

## Results

* Visualized sales trends using Matplotlib.
* Built a Linear Regression forecasting model.
* Predicted future sales successfully.
